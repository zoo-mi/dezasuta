私について

ー笑顔が紡ぐ魅力と創造性ー

笑顔は、言葉を超えて人々に喜びや温かさを
伝えることができる素晴らしい手段

笑顔には、人々を元気付け、つなげる力があります

笑顔には魅力があり、それが人々を引き付けたり、
魅了したりするだけでなく、創造性をも引き出す力がある

私の笑顔からお客様の笑顔、お客様の
笑顔からお客様のユーザーの方の笑顔
そのひとつひとつの笑顔を紡ぎ、
笑顔の連鎖で周りの人全てが
幸せになることを願っています。

使用ツール

  • Webデザイン

    お客様の課題、ユーザーのニーズや目標を理解しそれに基づいた使いやすいデザインを提供します

  • バナー

    魅力的で目を引くデザインを作成し、クライアントのメッセージを効果的に伝えます。

  • コーディング

    HTML、CSSを使用しクリーンで効率的なコードを作成し、ウェブサイトのスムーズな動作実現します。

  • 動画編集

    クライアントの要件やビジョンを理解し、それに合った魅力的な動画編集を提供します。

  • チラシ・ポスター

    クライアントの要件や目標を理解し、それに基づいた魅力的で効果的なデザインを提供します。

  • ロゴ

    レイアウトや配色、フォントの選定など、ビジュアル要素を使って名刺の情報を効果的に伝えます。

なんでwebデザイナーになったの？？

前職で、保育だよりや掲示物でレイアウトなどを考えることが好きで楽しく、どうすればもっと日中の子どもたちの姿を保護者の方々にわかりやすく伝わるだろう？？と考え始めました。そこからいろいろ調べていくうちにwebデザインに興味を持ちました。

あなたのアピールポイントは？？

なんと言っても”笑顔”です！！
笑顔は、コミュニケーションを円滑にし、人々を元気づける力があります。私は常に笑顔で接することを心がけており、明るく積極的な雰囲気を作り出すことができます。

どんなwebデザイナーになりたい？？

常に明るく笑顔でコミュニケーションを取り、“この人にお願いしたい”、”この人に頼んでよかった”“次もこの人にお願いしたい”と思ってもらえるwebデザイナーです。クライアントの方の悩みに寄り添い、思いを傾聴しながら,目的やビジョンを正確に理解した上で“クライアントの方と共に”解決していきます。
